planeta_engine::game::Componentを継承することで新しいコンポーネントを定義できる。

Componentには以下の仮想関数が定義されており、各々のタイミングでGameObjectから呼び出される。オーバーライドすることで各動作を定義できる。

bool NoUpdate()　ゲームオブジェクトにより更新されるかを返す。コンポーネントの登録時に呼び出され、falseならゲームオブジェクトの更新コンポーネントリストに登録される。登録後に値が変化しても反映はされないため、固定値が望ましい。
bool Initialize()　ゲームオブジェクトの初期化時に呼び出される。
bool Activated()　ゲームオブジェクトのシーン登録時に呼び出される。
void Update()　毎フレームゲームオブジェクト更新時に呼び出される。登録時にNoUpdate()==trueだった場合は呼び出されない。
bool InActivated()　ゲームオブジェクトのシーン登録解除時に呼び出される。
void Finalize()　ゲームオブジェクトの終了処理時に呼び出される。

通常関数は以下のものが定義されている。
std::shared_ptr<IGameObjectAccessor> game_object()　ゲームオブジェクトにアクセスする。
