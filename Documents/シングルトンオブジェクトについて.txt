自身のインスタンスをポインタで持つPointerSingltonで実装する。スタティックシングルトンは使わない。(メモリプール管理はスタティックシングルトンでいいかも)
関数をInstantiate,Initialize,Finalize,Dispose,instanceの五つの関数を持ち、インスタンスの作成破棄と、初期化終了処理を分離する。呼び出し惇は以下の通り。終了ｋ処理を行えば、もう一度初期化処理を行うこともできる。
static bool Instantiate():インスタンスを生成する。
*:InstantiateとInitializeの間で設定を行う。
bool Initialize():初期化を行う。
bool Finalize():終了処理を行う。
static bool Dispose():インスタンスを破棄する。

static C& instance():インスタンスの参照
static bool IsInstantiated():インスタンス化されているか
